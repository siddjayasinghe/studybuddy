# Study With Sidd

A distraction-free study video app with focus features.

## Prerequisites
- Node.js (v16 or higher)
- npm (comes with Node.js)

## Setup
```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser

## Features
- Curated study videos (lofi, rain sounds, pomodoro)
- Add your own YouTube videos
- Filter by category (Lofi, Rain, Nature, Pomodoro)
- Tab switch detection to keep you focused
- Dark/light mode
- Glassmorphism UI

## Tech Stack
React + Vite + Tailwind CSS
